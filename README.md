#Customer Data Visualization (Python)

This repository contains a simple Customer Data Analysis and Visualization project built using Python, Pandas, Matplotlib, and Seaborn.

#Files included
	•	individual_programs.py – Python programs for all visualizations
	•	Practice-Datasets.csv – Customer dataset
	•	README.md – Project documentation

#Features
	•	Age distribution analysis
	•	Gender percentage distribution
	•	City-wise customer analysis
	•	Monthly income analysis
	•	Spending score analysis
	•	Purchase trend visualization
	•	Product category analysis
	•	Relationship analysis using scatter plots

#Tools Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Seaborn
	•	VS Code

This project is created for academic learning and practice purposes.
